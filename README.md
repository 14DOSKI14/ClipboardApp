# Clipboard History Manager

En Clipboard history app som lagrer alt du kopierer og kategoriserer det automatisk som command, code eller text. Løser problemet med at Mac bare husker en ting i clipboard om gangen.

(Løser problemet med at macOS kun lagrer en ting i "clipboard", kopierer du noe nytt, er det gamle borte)

## Installer

pip install numpy matplotlib

## Kjør

1. cd src
   
2. python main.py

## Filer

- main.py - Hovedprogram med GUI
- data_handler.py - Lagring til JSON
- categories.py - Ordlister for kategorisering

## Bruk

- Klikk på en rad = kopierer til clipboard
- Dobbeltklikk = vis hele teksten
- stats = viser statistikk-graf
- clear = sletter all historikk

## Oppgavekrav

- Arrays - categories.py
- Vektoriserte beregninger - main.py
- If/else - main.py, categories.py
- For-løkker - main.py, categories.py
- Lese fra fil - data_handler.py
- Skrive til fil - data_handler.py
- Plotting - main.py
- Egendefinerte funksjoner - Alle filer
